Themantics

A simple Javascript and CSS theming engine.

Installation

Either download the theme.js and theme.css files straight into your project or link them straight from this page.


Link

Add these lines to your HTML head tag.

            <script src="https://themarstonconnell.github.io/Themantics/theme.js">
</script><link rel="stylesheet" href="https://themarstonconnell.github.io/Themantics/theme.js">

Download

Right click on the links to save the files.
